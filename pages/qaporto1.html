Team Automation from Bootcamp Program

As a participant in the Katalon Automation Testing Bootcamp Program, I specialized in web, mobile, and API testing, gaining proficiency in automated testing across multiple platforms. Working in a collaborative team of two, we harnessed the power of Katalon Studio to create robust test scripts, ensuring the quality and reliability of our software applications. We leveraged GitLab for version control, enabling seamless collaboration and code management.

Additionally, we utilized Google Spreadsheets for efficient progress reporting, ensuring transparency and effective communication throughout the testing process. This program empowered me with the skills to conduct comprehensive test automation in a collaborative and organized manner, resulting in high-quality software delivery.
